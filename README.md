# Move in Peace
A Flutter mobile application designed to promote healthy work habits by providing customizable timer reminders for stretching and movement breaks during different activities.

## Usage Guide
1. **Select Your Mode**: Choose from Study, Work, or Meeting mode on the home screen
2. **Set Timer Duration**: Adjust the timer length using the preset buttons
3. **Start Session**: Begin your focused work period with the start button
4. **Receive Reminders**: Get notifications when it's time to take a break
5. **Follow Exercises**: Use the recommended stretches from the exercise library
6. **Track Progress**: View your session history and improvement analytics
